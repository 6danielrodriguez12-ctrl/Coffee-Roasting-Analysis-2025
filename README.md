# Coffee Roasting Analysis 2025
Analysis of roasting weight loss and annual production efficiency.

## 📝 Project Overview
This project analyzes a full year of coffee roasting production to identify efficiency patterns and reduce weight loss (shrinkage). I transformed raw, unorganized data into actionable business insights.

## 🚀 Key Business Insights
* **Peak Efficiency:** The highest efficiency was achieved in **June**, with a weight loss of only **2.78%**.
* **Year-end Trend:** A significant increase in weight loss (up to **11.4%**) was observed from October to December.
* **Operational Impact:** This trend suggests the need to review roasting profiles or green bean moisture levels during the final quarter.

## 🛠️ Tech Stack
* **Excel:** Data cleaning and initial structuring.
* **MySQL:** Data consolidation (UNION ALL) and business logic implementation.
* **Tableau:** Interactive trend visualization.
